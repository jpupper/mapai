# DiploIA / Comunidad - Plataforma de Mapas

Este repo sirve dos experiencias:

- **DiploIA** (contenido de la diplomatura): mapas de herramientas, nube de universos, tutoriales, presentación, etc.
- **Comunidad** (root): registro/login y creación + exploración de mapas creados por usuarios.

## Variables de Dirección (URL Parameters)

Podés modificar el comportamiento de las páginas utilizando los siguientes parámetros en la URL:

*   **`fullscreen=true`**: (En `mapadeherramientas.html`) Inicia el mapa en modo pantalla completa automáticamente.
*   **`node=[id]`**: (En `presentacion.html`) Carga directamente un componente o tecnología específica por su ID. El ID se actualiza automáticamente en la barra de direcciones al navegar.

## Accesos

Accesos típicos:

*   Comunidad (root): `https://example.com/`
*   DiploIA (subdominio): `https://diploia.example.com/`
*   MapAI (prefijo en dominio principal): `https://example.com/mapai/`
*   Legacy (compatibilidad por prefijo): `https://example.com/diploia/`
